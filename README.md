# WHO Life Expectancy: EDA + Linear Regression Insights

Exploratory data analysis and a predictive linear regression model built on 16 years (2000–2015)
of WHO/UN data covering 193 countries — examining how mortality, immunization, economic
development, and education relate to life expectancy.

## Highlights
- Full EDA: missing value analysis, distributions, developed vs developing comparison, time trends
- Correlation analysis across 20+ health, economic, and social indicators
- Linear regression model — **R² = 0.82**, RMSE ≈ 3.9 years
- Coefficient interpretation to identify the strongest drivers of life expectancy

## Dataset
[Life Expectancy (WHO)](https://www.kaggle.com/datasets/kumarajarshi/life-expectancy-who) — compiled
from WHO Global Health Observatory and UN data by Deeksha Russell and Duan Wang.
